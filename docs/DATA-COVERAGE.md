# 解放條件資料覆蓋率（unlocks.json）

> 產出：`node scripts/import-unlocks.mjs`；驗證與本報告數據：`node scripts/validate-unlocks.mjs`。
> 資料基準：Kiranico 破曉 TU5（Ver16）、任務清單四軌（村莊／集會所初階／進階／Master）。
> 最後更新：2026-07-07。

## 信心度定義

| 信心度 | 意義 | 來源 |
|---|---|---|
| `confirmed` | 已確認的遊戲常數 | TU／後期魔物的 MR 等級解放門檻（Game8 攻略確認：月迅龍等 MR10、激昂金獅子 MR50、嗟怨轟天怨虎龍 MR100、傀異克服系 MR110~180） |
| `inferred` | 任務星級/素材推導 | (1) 裝備 `sourceMonster` ×「魔物首次出現星級」映射，依裝備階級取對應軌道（村裝→村/低位、HR 裝→上位、MR 裝→Master）；(2) 傀異素材門檻（見下節） |
| `unverified` | 稀有度近似 | 無來源魔物、無傀異素材可推導，退回 rarity→里程碑映射 |

## 傀異素材門檻（anomaly-material 推導）

R9+ 的 MR 裝備逐件掃描 Kiranico 詳細頁的**生產＋強化素材**：素材含傀異素材者，依 [Game8 傀異任務指南](https://game8.co/games/Monster-Hunter-Rise/archives/381480)確認的等級表設定 MR 門檻（共 597 件，佔原 unverified 大宗——「無法歸類」的 R10 終端武器多為傀異素材把關）：

| 傀異素材 | 任務等級 | 門檻 |
|---|---|---|
| 傀異化之骨／之皮 | A1★ | MR10 |
| 傀異化血液／龍骨 | A2★ | MR20 |
| 傀異化甲殼／鱗片 | A3★ | MR30 |
| 傀異化之牙／之爪 | A4★ | MR50 |
| 傀異化凶系（骨/角/鱗/殼/爪/翼/龍血/牙） | A5★~A7★ | MR50 下限＋note 標註研究等級（Lv81/111/181） |
| 傀異調查票劵 | 傀異調查任務 | MR10＋note |

規則：武器的「生產」與「強化」是兩條取得路徑，**任一路徑不需傀異素材即不設門檻**（取路徑門檻最小值）；A5★+ 由傀異研究等級把關、無對應 MR 軸，故記 MR50 下限並以 `note` 欄誠實標註（不假裝精確）。

驗證項目（`validate-unlocks.mjs`，全數通過）：引用完整性（雙向）、每條目至少一個里程碑軸、信心度合法值、星級範圍、inferred 與階級零矛盾。對照驗證：進度時間軸單調遞增、全解進度（村6/集8/M6/MR200）下 5544 件 100% 可製作。

## 名稱覆寫表（社群回報修正機制）

變種／傀異克服魔物的素材名稱前綴與魔物名單不一致時，來源魔物啟發式會誤判為基底魔物，造成解放章節嚴重低估。`import-unlocks.mjs` 的 `NAME_MONSTER_OVERRIDES` 以裝備名稱樣式強制指定來源，目前收錄（2026-07-07 社群實測回報後修正）：

| 名稱樣式 | 實際來源 | 解放 |
|---|---|---|
| 赫耀・曆 | 天彗龍（傀異克服） | MR160 |
| 霞龍醒 | 霞龍（傀異克服） | MR110 |
| 脈動鋼龍／脈動帝王 | 鋼龍／炎王龍（傀異克服） | MR120／140 |
| 混沌黑蝕、「○or○」武器 | 混沌黑蝕龍 | MR10（需與騎士帕茲對話觸發緊急任務） |
| 鎧怨鬼／鎧幽鬼／禍鎧・怨 | 嗟怨轟天怨虎龍 | MR100 |
| 鳴神○○真 | 百龍之源雷神龍 | M6★ |

| 怒天 | 激昂金獅子 | MR50 |
| 矜持 | 紅蓮爆鱗龍 | MR10 |

樣式刻意收窄以免誤傷基底系列（禍鎧・真 R8 是一般 MR 怨虎龍裝 M3★、混沌之弓是奇怪龍武器、雷鳴神是一般雷神龍武器——皆已驗證不受影響）。

## 已知限制

- 推導鏈上有兩層啟發式：來源魔物判定（詳細頁素材佔比 ≥60% 或名稱含魔物名）、任務目標句魔物比對（最長名優先＋遮蔽）。「狩獵所有目標」類任務未列名，不參與統計。
- 特殊解放套裝（活動任務、支線委託、大師之魂類）無法從任務星級推導，會落在 unverified。
- unverified 的 rarity→里程碑映射刻意保守但仍是近似：R8→M1、R9→M3、R10→M5。

## 待補清單（優先序）

1. **preset 用裝人工校驗**：68 套 preset 實際會推薦的裝備數量有限、又是新手真正看到的，逐件人工驗證後升 confirmed，投報率最高。
2. **inferred 抽樣校驗**：對照 Game8／老玩家經驗抽查各階級樣本，修正推導錯誤（社群回報從這裡進）。
3. **特殊解放套裝逐件標註**：活動／支線／稱號解鎖類（皮製系列以外的 unverified 大宗）。
4. **傀異研究等級細分**：R10 傀異強化素材實際受傀異研究 Lv 與 MR 雙重門檻，目前僅以 Master★／MR 等級近似。

## 回報錯誤

發現某件裝備的解放條件標錯？請開 GitHub issue 附上：裝備名稱、遊戲內實際的解放時機（第幾星／MR 多少）、（可選）出處截圖。單筆修正會以人工覆寫表的形式併入匯入腳本並升為 confirmed。
