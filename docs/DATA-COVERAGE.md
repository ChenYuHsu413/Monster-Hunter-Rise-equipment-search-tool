# 解放條件資料覆蓋率（unlocks.json）

> 產出：`node scripts/import-unlocks.mjs`；驗證與本報告數據：`node scripts/validate-unlocks.mjs`。
> 資料基準：Kiranico 破曉 TU5（Ver16）、任務清單四軌（村莊／集會所初階／進階／Master）。
> 最後更新：2026-07-07。

## 信心度定義

| 信心度 | 意義 | 來源 |
|---|---|---|
| `confirmed` | 已確認的遊戲常數 | TU／後期魔物的 MR 等級解放門檻（Game8 攻略確認：月迅龍等 MR10、激昂金獅子 MR50、嗟怨轟天怨虎龍 MR100、傀異克服系 MR110~180） |
| `inferred` | 任務星級推導 | 裝備 `sourceMonster` ×「魔物首次出現星級」映射；依裝備階級取對應軌道（村裝→村/低位、HR 裝→上位、MR 裝→Master） |
| `unverified` | 稀有度近似 | 無來源魔物可推導（礦石裝、雜貨裝、混合素材裝），退回 rarity→里程碑映射 |

## 已知限制

- 推導鏈上有兩層啟發式：來源魔物判定（詳細頁素材佔比 ≥60% 或名稱含魔物名）、任務目標句魔物比對（最長名優先＋遮蔽）。「狩獵所有目標」類任務未列名，不參與統計。
- 特殊解放套裝（活動任務、支線委託、大師之魂類）無法從任務星級推導，會落在 unverified。
- unverified 的 rarity→里程碑映射刻意保守但仍是近似：R8→M1、R9→M3、R10→M5。

## 待補清單（優先序）

1. **preset 用裝人工校驗**：68 套 preset 實際會推薦的裝備數量有限、又是新手真正看到的，逐件人工驗證後升 confirmed，投報率最高。
2. **inferred 抽樣校驗**：對照 Game8／老玩家經驗抽查各階級樣本，修正推導錯誤（社群回報從這裡進）。
3. **特殊解放套裝逐件標註**：活動／支線／稱號解鎖類（皮製系列以外的 unverified 大宗）。
4. **傀異研究等級細分**：R10 傀異強化素材實際受傀異研究 Lv 與 MR 雙重門檻，目前僅以 Master★／MR 等級近似。

## 回報錯誤

發現某件裝備的解放條件標錯？請開 GitHub issue 附上：裝備名稱、遊戲內實際的解放時機（第幾星／MR 多少）、（可選）出處截圖。單筆修正會以人工覆寫表的形式併入匯入腳本並升為 confirmed。
